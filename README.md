## Getting Started

Follow these steps to set up and run the project locally:

`git clone https://github.com/Ibrahim-Rezq/minimal-portfolio`

`cd minimal-portfolio`

`npm install`

`npm run dev`

Visit `http://localhost:3000` in your browser to see the portfolio.
